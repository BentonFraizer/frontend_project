За недостатком времени отправляю задание в таком виде.

// Эндпоинты для сущности "Пользователи"
GET    - "/users"     - получить всех пользователей
GET    - "/users/:id" - получить одного пользователя
POST   - "/users"     - создать нового пользователя
PUT    - "/users/:id" - изменить одного пользователя
DELETE - "/users/:id" - удалить одного пользователя


Для успешного ответа сервера подойдёт статус 200 - ОК (Запрос успешно выполнен)
Для успешного создания нового пользователя подойдёт стату 201 - Created
При отсутствии авторизации подойдёт статус 401 - Unauthorized

Посмотрел остальные статусы, что-то не счёл другие подходящими для данного задания.
